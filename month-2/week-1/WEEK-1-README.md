# Month 2, Week 1 Project
## Objective
Build logic to solidify understanding of these JavaScript concepts: Variables and Data types (including Array & Object), Data Manipulation for different typees of data, Loops and Conditionals.

## Project

Title: Grocery Store Inventory Management
Challenge: Given an the store's inventory which is an array of objects, you are to process the data to answer the following questions as the manager:

1. Task 1 - What items are out of stock?
2. Task 2 - What items are on discount?

## Guide
- Do not create a new file, see index.js file in `month-2/week-1/` folder.
- Follow the instructions in the file to come up with a solution. You will defend your solution, ensure it is yours.
- Run this command: `node month-2/week-1/index.js` in your terminal to see `console.log()` outputs which is the result of calling your function.
